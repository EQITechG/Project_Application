STEPS
1.  CHOOSE SERVER IP FROM DROP DOWN LIST ON CONNECTION TAB
2.  CHOOSE ANY FREE PORT
3.  CLICK THE START SERVER BUTTON TO START SERVER
4.  PROVIDE SERVER IP ADDRESS AND PORT TO ANY CLIENT(S) YOU WISH TO INITIATE CONNECTION WITH
5.  ANY CONNECTED CLIENT WILL APPEAR ON CLIENT LIST
6.  FOR CLIENT AND SERVER EVENTS, VIEW SERVER LOGS
7.  DISABLE SERVER IF REQUIRED (IT WILL DISABLE ANY RUNNING CLIENTS)
8.  SAVE SERVER LOGS IF REQUIRED
9.  THANK YOU :D
