Introduction

Morse Code was developed by Samuel Morse in 1832 for use with the telegraph system.
In Morse Code, each letter of the alphabet and numerals are represented by a combination
of dots and dashes. In a sound-oriented systems, the dot represents a short sound while
the dash represents a long sound.

How to Use

Convert from English to Morse Code

  1. Type the English sentence that you would like to convert in the left-hand text area.
  2. Press the 'Convert to Morse Code' button.


Figure 1 Convert from English to Morse Code

Note: Each Morse-coded letter is separated by a space and each Morse-coded word is
separated by three spaces.

Convert from Morse Code to English

  1. Type the Morse Code that you would like to convert in the right-hand text area.
    Separate each Morse-coded letter by a space and each Morse-coded word by
    space-\w-space.
  2. Press the 'Convert to English' button.


Figure 2 Convert from Morse Code to English
